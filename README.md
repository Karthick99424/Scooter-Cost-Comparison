# 🛴 Scooter Cost Comparison

## Smart 14-Year EV vs Petrol TCO Dashboard

This interactive web application provides a comprehensive total cost of ownership (TCO) comparison between electric scooters and petrol-powered scooters over a 14-year period.

### 🚀 Live Demo

Visit the live dashboard: [https://Karthick99424.github.io/Scooter-Cost-Comparison/](https://Karthick99424.github.io/Scooter-Cost-Comparison/)

### ✨ Features

- **Interactive Dashboard**: Real-time TCO calculations with customizable parameters
- **14-Year Comparison**: Long-term cost analysis for both EV and petrol scooters
- **Itemized Breakdown**: Detailed costs including:
  - Purchase price
  - Charging/Fuel costs
  - Maintenance costs
  - Insurance
  - Registration and other fees
- **Visual Charts**: 
  - Annual cost comparison
  - Cumulative cost over 14 years
  - Cost breakdown pie charts
- **Vehicle Presets**: Pre-configured popular scooter models
- **Responsive Design**: Works on desktop, tablet, and mobile devices

### 📊 What You Can Calculate

- **Fuel/Charging Costs**: See the difference between electricity vs petrol
- **Maintenance**: Compare maintenance expenses over 14 years
- **Insurance**: Account for insurance cost differences
- **Registration Fees**: Include registration and other recurring costs
- **Total Ownership Cost**: Get the complete financial picture

### 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables
- **JavaScript**: Interactive calculations and DOM manipulation
- **Chart.js**: Beautiful, responsive charts
